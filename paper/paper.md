# omicsmeta: Automated Harmonization of Public Omics Metadata

## Summary

`omicsmeta` is an open-source Python package for converting messy public omics
metadata into analysis-ready ontology-mapped tables.

## Statement of Need

Public GEO, SRA, and BioSample metadata often encode the same biological
concept with inconsistent field names, abbreviations, typos, and free text.
This slows cross-study meta-analysis and makes manual curation difficult to
scale.
